# Directory Structure

- `config/`: application configuration.
- `data/design/`: static design decision data.
- `src/`: new integration code.
- `design_tool/`: migrated design engine compatibility package.
- `steps/`, `tools/`, `orchestrator.py`: migrated DevFlow runtime.
- `ucos/`: cognitive memory system.
- `workspace/`: user data and generated outputs.
- `_archive/`: references to legacy source projects.
