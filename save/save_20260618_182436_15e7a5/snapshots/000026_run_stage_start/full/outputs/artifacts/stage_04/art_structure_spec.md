# Art Structure Spec

- Project type: Unity
- Every art item keeps an `asset_id` and source reference.
- Config assets are treated as data-facing production assets.
- Missing style or audio requests stay out of scope unless current project docs add them.

## Allowed Roots

- Assets/Art/
- Assets/UI/
- Assets/VFX/
- Assets/Audio/
- Assets/Textures/
