# Claude Code 更新工作流程

在更新 Claude Code 之前，用户询问如何安全更新。

**Why: 避免更新导致工作丢失**
- 确保所有工作已提交到 Git
- 确保重要决策已保存到记忆
- 确认会话状态可以安全恢复

**How to apply: 更新前的检查清单**

1. ✅ 检查 Git 状态 - 确保没有未提交的修改
2. ✅ 保存重要记忆 - 记录未完成的工作
3. ✅ 记录当前上下文 - 如果有未完成的任务
4. ✅ 关闭应用并更新
5. ✅ 重启后验证环境

**What to remember: 更新后需要验证**
- 项目路径是否正确
- Git 仓库状态
- 记忆系统是否正常
- 环境变量和配置

与 [[项目工作流]] 相关。
